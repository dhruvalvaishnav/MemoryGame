## 🎮 How to Play

1. Cards are placed face-down on the board
2. Click any card to flip it and reveal its emoji
3. Click a second card - if they match, they stay face-up
4. If they don't match, both flip back after a short delay
5. Match all pairs to win!

---

## ⚙️ Features

- 🃏 Randomized card layout on every game
- ⏱️ Move counter to track your performance
- 🔄 Restart button to play again
- 📱 Responsive design - works on mobile and desktop
